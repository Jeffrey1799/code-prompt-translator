v0.1.6 版本记录：
- 重构侧边栏 UI 布局：将中文 Prompt 输入框（Chinese Prompt）吸底排列在侧边栏最下方，与右侧 Agent CLI 命令行输入框保持水平平齐，大幅缩短鼠标左右交互直线距离。

v0.1.5 版本记录：
- 支持斜杠技能前缀自动保留（Slash Command Preservation）：自动检测并提取句首以 `/<skill-name>` 开头的斜杠指令，仅对后续中文描述进行翻译并精准还原合成，防止 CLI 技能命令被误翻译。

v0.1.4 版本记录：
- 修复终端交互问题：改用终端内括号粘贴机制（`workbench.action.terminal.paste`），解决带换行符或包含追加指令的 Prompt 填入终端时引发的误自动回车发送及多行指令丢失问题。

v0.1.3 版本记录：
- 优化人机交互与无缝全键盘工作流：翻译后自动将英文 Prompt 填充至当前活跃终端（Terminal），并自动将键盘焦点切换至终端（不自动回车执行，便于确认后按回车提交）。
- 支持在 Webview 选项与 VS Code 设置中开启/关闭“Send to active terminal & focus”功能。
- 更新输入框 placeholder 提示信息为英文快捷键提示。

v0.1.2 版本记录：
- 优化输入框交互逻辑：在输入框内输入内容后直接按 Enter 键触发翻译与复制（Translate & Copy）。
- 支持 Shift+Enter 快捷键在输入框中进行换行。
- 更新状态栏底部的快捷键提示信息。
