# Changelog

## 0.1.0

- Initial release.
- Add Activity Bar container and sidebar WebviewView UI.
- Add faithful Chinese-to-English coding prompt translation through OpenAI-compatible Chat Completions API.
- Add SecretStorage API key management.
- Add VS Code settings for Base URL, model, temperature, append instruction, auto-copy, and timeout.
- Add local appending of `Please respond and communicate with me in Chinese.`.
- Add clipboard copy support through `vscode.env.clipboard.writeText`.
- Add command palette commands for open, translate, set/clear API key, copy output, and clear.
